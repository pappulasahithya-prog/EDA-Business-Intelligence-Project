# EDA Business Intelligence Project

## Overview
This project performs Exploratory Data Analysis (EDA) on the Superstore dataset to identify business trends, analyze sales and profit, and provide data-driven recommendations. The project also includes statistical validation using Pearson Correlation to verify the relationship between Discount and Profit.

## Objectives
- Analyze sales and profit performance.
- Identify trends across product categories and regions.
- Discover factors affecting profitability.
- Validate business insights using statistical analysis.
- Present findings through data storytelling.

## Dataset
- Dataset: Superstore Dataset
- Features include:
  - Order ID
  - Product Category
  - Sales
  - Profit
  - Discount
  - Region
  - Customer Information

## Key Insights

- Higher discounts reduced profitability.
- Regional performance varied across the business.
- Data-driven decisions can improve business performance.

## Statistical Validation

**Test Used:** Pearson Correlation Test

### Hypothesis

**Null Hypothesis (H₀):**
Discount has no significant effect on Profit.

**Alternative Hypothesis (H₁):**
Discount has a significant effect on Profit.

### Results
- Correlation Coefficient: **-0.4578**
- P-value: **0.0**

### Conclusion
The P-value is less than 0.05, so the Null Hypothesis is rejected.

There is a statistically significant negative relationship between Discount and Profit. Higher discounts generally reduce profitability.
